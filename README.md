Patient Safety Portal

A full-stack web application for monitoring and managing patient safety indicators across various health conditions such as **Diabetes, Cardiac Disorders, Maternity, and more**.

 Project Overview
This portal helps healthcare professionals visualize and track patient safety metrics in real time. The dashboard provides an at-a-glance view of patient data, condition-specific insights, and safety alerts â€” enabling timely interventions and improved care outcomes.

 Tech Stack
- Frontend:React with TypeScript
- Backend: C# (ASP.NET Core Web API)
- Database:SQL Server
- State Management: React Context API
- Styling:** Material-UI with custom CSS
- Authentication: JWT-based authentication

 Key Features
- Interactive Dashboard:** Charts and tables displaying patient safety metrics
- Patient Profiles:** Detailed views of individual patient records
- Condition Tracking:**
  - Diabetes monitoring (glucose levels, HbA1c trends)
  - Cardiac disorder indicators (blood pressure, cholesterol)
  - Maternity care tracking (prenatal visits, risk factors)
- Safety Alerts:** Notifications for abnormal values or high-risk patients
- Role-Based Access:** Different views for doctors, nurses, and admins
- Responsive Design:** Works on desktop and tablet devices
